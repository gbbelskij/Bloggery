# Bloggery Service API

## О проекте

Это проект сервиса для ведения блогов, реализованный на Go с использованием Gin фреймворка. В проекте реализована авторизация через JWT с использованием Redis для управления сессиями и blacklist токенов. Пользователи могут создавать, просматривать, редактировать и удалять свои посты, с поддержкой пагинации.

### Основные возможности
- Авторизация и аутентификация через JWT
- Хранение постов и пользователей в Postgres
- Хранение и проверка токенов в Redis (черный список для логаута)
- CRUD операции с постами блога
- Пагинация и фильтрация постов
- Middleware для проверки авторизации
- Запуск Redis в Docker контейнере для кэширования и управления сессиями

## Установка и запуск

```bash
git clone https://github.com/gbbelskij/Bloggery
cd Bloggery
docker-compose up --build
```

### Предварительные требования
- Go 1.20+
- Docker и Docker Compose
